### Requirements:
shell with python3 


### Usages:

```
# get the code
git clone https://github.com/creel904/beginner-531-text-log.git`
cd beginner-531-text-log

# run program (enter your 3-week cycle 1RM or Training Maxes for each main lift):
python3 beginner531.py -s 255 -b 185 -d 225 -p 115

# or, alternatively:
python3 beginner531.py --squat=255 --bench=185 --deadlift=225 --press=115

# to show help
python3 beginner531.py -h
python3 beginner531.py --help
```
